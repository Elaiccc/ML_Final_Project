Fake News Detector & Generator

An end‑to‑end pipeline for generating synthetic fake news and detecting it with ML models.

Problem Overview

Automatically generate high‑quality fake news using GPT, then train a classifier to distinguish real vs. fake articles—important for misinformation research and detection.

Data

• Real news corpus of X articles
• 15,000 synthetic fake samples via OpenAI API
• TF‑IDF vectorization on text

Model

• Logistic Regression on TF‑IDF features
• Hyperparameters tuned via grid search
• Achieves ROC‑AUC of 0.99 on held‑out test set

Results

Precision: 0.98  |  Recall: 0.97  |  F1‑Score: 0.975  |  ROC‑AUC: 0.99
